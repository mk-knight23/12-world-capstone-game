# World Analyzer // MK OS-X

An immersive, data-driven geographic exploration experience with a cyberpunk aesthetic. Browse countries, view statistics, and explore global data through a sophisticated terminal-style interface.

## Features

- Interactive country list with search/filter
- Real-time data from REST Countries API
- Cyberpunk/MK OS-X visual theme
- Scanline and grid background effects
- Responsive design for all screen sizes
- Zero dependencies - pure HTML, CSS, and JavaScript

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, animations, flexbox, grid
- **JavaScript** - Vanilla ES6+ with native fetch API
- **Data** - REST Countries API (https://restcountries.com)

## Quick Start

Simply open `index.html` in any modern browser:

```bash
open index.html
```

## Deployment

Deploy `index.html` to any static hosting:
- GitHub Pages
- Netlify (drag & drop)
- Vercel
