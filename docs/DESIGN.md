# Design System: 12-world-capstone-game

## Visual Philosophy
The design follows a **Cyberpunk MK_OS-X / Command Center** aesthetic. It prioritizes neon cyan accents against a deep slate void (#020617), using scanlines and grid patterns to simulate a technical data terminal.

## Design Patterns
-   **Cyan Scanning**: Persistent neon pulse on the primary globe icon and search boundaries.
-   **Cyber Borders**: Sharp, tech-themed borders with partial gradients to emphasize data units.
-   **Scanline Overlay**: A persistent CSS animation layer simulating CRT terminal interference.
-   **Technical Muted Typography**: High use of monospaced fonts with variable opacity for "secondary data" feel.

## Color Palette
-   **OS Void**: Slate-950/Midnight (#020617)
-   **Technical Primary**: Cyan-500 (#06b6d4)
-   **Data White**: Zinc-50 (#fafafa)
-   **Alert State**: Amber-400 (Selective highlights)

## Interaction Design
-   **Probe Initialization**: Animated scale-in/out for country statistics cards.
-   **Infinite Grid**: Fixed background grid pattern that provides spatial depth.
-   **Analytical Motion**: Rapid cross-fades and slide-ins for entity switching to simulate high-speed data access.
